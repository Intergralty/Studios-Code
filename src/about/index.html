About

CyCode is a HTML editor, Markdown Editor, and WYSIWYG Editor with a primary focus on web development. It was created and developed by Cyril John Magayaga, I'm 14 years old and Filipino student (Speech and language impairment).

It is formerly known as Studios Code or Studios One. It was first released on December 23, 2021 for Windows. CyCode is available for the macOS, Windows, and Linux (Android and Linux Distros) operating systems.
